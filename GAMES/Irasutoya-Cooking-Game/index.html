Irasutoya Cooking Game

A browser game made with irasutoya-style illustrations.

Project Type

Browser Game

Tools

Unity, GPT-4

About

This project was inspired by my appreciation for the charming visual style of Irasutoya. After discovering its distinctive and cute illustration aesthetic, I used ChatGPT to generate game assets that emulate this style, and then built a browser-based mini game in Unity based on those materials. The process involved exploring visual direction, generating and organizing assets with AI, and implementing gameplay mechanics before deploying the final project to the web. It represents an experimental integration of generative AI tools with practical game development.

Visual Style Reference

The visual style is inspired by Irasutoya (いらすとや), a Japanese free illustration collection known for its cute and distinctive aesthetic.
